## How to Build and Run

First, clone the `Acai` project:

```
$ git clone https://github.com/xlab-uiuc/acai.git
```

Second, install the `Acai` project in your local maven repository:

```
$ cd [path-to-acai]
$ mvn clean install
```

Third, compile the project:

```
$ cd [path-to-ccc]
$ mvn compile
```
